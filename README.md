# doclens

An AI system that helps users understand and explore documents by finding relevant information and showing evidence.

## Libraries Used

- **faiss-cpu** — vector similarity search library used to index and retrieve document embeddings.
- **groq** — client for accessing Groq's fast LLM inference API.
- **langchain-groq** — LangChain integration for using Groq models within LangChain pipelines.
- **PyPDF2** — reads and extracts text content from PDF files.
- **langchain_google_genai** — LangChain integration for Google's Generative AI models.
- **langchain** — framework for building LLM-powered applications with chains, agents, and retrieval.
- **streamlit** — builds the interactive web UI for the application.
- **langchain_community** — community-maintained LangChain integrations and utilities.
- **python-dotenv** — loads environment variables from a `.env` file.
- **pypdf** — alternative PDF reading/parsing library used for text extraction.
